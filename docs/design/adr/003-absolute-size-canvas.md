# ADR-003: Absolute Size Canvas for Reproducibility

## Context
Infinite canvases (like OneNote/Excalidraw) cause issues when printing or submitting academic/professional documents (A4/B5), leading to broken layouts or clipped content.

## Decision
We enforce an **Absolute Physical Coordinates** model:
1.  Notebooks use a strict Millimeter-based coordinate system.
2.  Logical "Page" boundaries (e.g., A4: 210x297mm) are rendered as visual guides but do not exist as structural nodes.
3.  Coordinates are relative to the notebook origin (or local structural parent).
4.  PDF export is generated by partitioning the infinite canvas into pixel-perfect logical pages.

## Status
Accepted

## Consequences
- **Positive**: Reliable PDF output; predictable margins; easier to organize content into logical "pages".
- **Negative**: Less freedom for "infinite brainstorming" compared to competitive apps.
