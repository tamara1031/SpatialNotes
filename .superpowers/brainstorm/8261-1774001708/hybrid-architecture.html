Strategic Implementation Detail: MM-to-Pixel Mapping

The "Core" is MM

All positions in Yjs are stored in physical millimeters (A4: 210x297). This ensures PDF export is always pixel-perfect.

The "View" is Pixels

A Matrix3x3 transform converts MM to Screen Pixels based on the current Zoom level. We apply this transform via ctx.setTransform() for the canvas layers.
