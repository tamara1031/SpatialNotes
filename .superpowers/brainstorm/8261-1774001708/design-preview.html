Strategy: Custom Ink API Engine (Option A)

The "Magic Desk" foundation for native-level responsiveness.

1. Low-Latency Trail

Uses DelegatedInkTrailPresenter. The browser hands off rendering to the OS compositor, bypassing the JS event loop for the "active" stroke.

2. Pressure Sensitivity

Captures PointerEvent.pressure (0.0 to 1.0). Maps pressure to line width in real-time for an organic, "fountain pen" feel.

3. Spline Smoothing

Transforms raw jagged points into smooth curves using Catmull-Rom interpolation after the pen is lifted.

Canvas Layer Stack (Side View)
Interaction Layer: Selection boxes, Tape, UI Handles
Active Layer (Ink API): Real-time trailing ink
Static Layer: Committed Strokes, Images, PDF Background

The Pipeline

  1. PointerDown: Initialize StrokeBuffer and InkTrail.
  2. PointerMove: Update InkTrail (Immediate) + Save points (Buffer).
  3. PointerUp: Apply Smoothing + Simplification -> Create StrokeElement Command.
